Opgave Oefening: tafels

  • Maak een webpagina aan met een label, inputfield en button
  • Bij klikken op het label krijgt inputfield de focus
  • Bij klikken op de button "Go!" wordt onderaan in een div dynamisch de tafel van vermenigvuldiging van het ingetikte getal toegevoegd
  • Controle of de ingevoerde value in het inputfield effectief een nummer is. Zoniet, alert met foutmelding tonen
  • 1 tafel ziet er als volgt uit
  • Header: Tafel van x aangemaakt op: [tijdstip waarop op de button Go! is gedrukt]
  • Enkel tijdstip in header, niet de datum
  • Alle even rijen hebben een grijze achtergrond
  • Breedte: 300px
  • Geen <table> tag gebruiken, maar <div>'s
  • Na toevoegen van een tafel moet inputfield terug leeg gemaakt worden
  • Bouw na iedere ingave alle tafels opnieuw op en toon ze op het scherm
    • // handige functie om alle children van een element te verwijderen
// verwijder alle childs van een element
const verwijderAlleChildren = (element) => {
    while (element.firstChild) {
        element.removeChild(element.firstChild);
    }
}
}
